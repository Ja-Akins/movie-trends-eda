# Movie Trends EDA
A simple exploratory data analysis project that visualizes movie trends by genre, year, and average revenue using Python (Pandas, Matplotlib, Seaborn).

## Features
- Top movie genres by count and revenue  
- Yearly movie trends  
- Interactive visualizations (optional via Streamlit)

## Tech Stack
Python, Pandas, Matplotlib, Seaborn, Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
